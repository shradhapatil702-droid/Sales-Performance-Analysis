## Overview
This project analyzes sales data to evaluate overall business performance, identify revenue trends, and understand key factors influencing sales using Python and data visualization techniques.

## Objectives
- Analyze total sales and revenue trends
- Identify top-performing products and regions
- Study monthly and yearly sales performance
- Generate insights to support business decisions

## Dataset Description
The dataset consists of sales transaction records with the following columns:
- Order Date
- Product
- Category
- Region
- Sales
- Quantity
- Profit

## Data Preprocessing
- Converted date column to datetime format
- Removed duplicates and handled missing values
- Converted numeric columns to proper data types
- Cleaned and structured data for analysis

## Exploratory Data Analysis
- Overall sales and profit analysis
- Category-wise and region-wise sales performance
- Monthly sales trends
- Top products by sales and profit

## Visualizations
- Sales Trend Over Time (Line Chart)
- Category-wise Sales (Bar Chart)
- Region-wise Sales Distribution
- Top Products by Revenue

## Key Insights
- Certain categories contribute most to total revenue
- Sales show seasonal trends across months
- A small number of products drive major sales
- Regional performance varies significantly

## Technology Stack
- Programming Language: Python
- Libraries: Pandas, Matplotlib
- Environment: Jupyter Notebook
